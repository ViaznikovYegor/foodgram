# Foodgram

**Foodgram** — это сервис для публикации и хранения рецептов с возможностью добавления рецептов в избранное и формирования списка покупок.

## Описание проекта

Проект позволяет пользователям:
- Регистрироваться и авторизовываться
- Публиковать рецепты с ингредиентами и тегами
- Добавлять рецепты в избранное
- Формировать список покупок
- Подписываться на других авторов

## Используемые технологии

### Backend
- Python 3.12
- Django 5.1
- Django REST Framework
- PostgreSQL
- Gunicorn
- drf-spectacular (OpenAPI)

### Инфраструктура
- Docker
- Docker Compose
- Nginx
